# Tic Tac Toe AI

A web app for playing Tic Tac Toe against an AI.

## Setup
1. Clone the repo.
2. Run `npm install`.
3. Create a `.env` file with your `GROQ_API_KEY` and `PORT` (e.g., `PORT=3000`).
4. Run `npm start` or `npm run dev`.

## Features
- User accounts
- Tic Tac Toe vs AI
- Leaderboard and stats
